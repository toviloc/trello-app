# ReactJS + Hooks + Redux - Building an app like Trello from scratch

## Requirements

- \*\*nodejs >= v16.15.1
- \*\*npm >= v8.18.0
- \*\*yarn >= v1.12.19

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ yarn install
$ yarn start
```
